Aplikasi Manajemen Karyawan Sederhana
Di buat dengan Netbean IDE 8.2


Fitur :
-Login Multiuser 
-Manajemen Karyawan
-Manajemen User
-Manajemen Gaji 
-Cetak Laporan Gaji (IReport)


Aktifkan XAMPP
Buat database di localhost dengan nama db_karyawan, kemudian import db_karyawan.sql
